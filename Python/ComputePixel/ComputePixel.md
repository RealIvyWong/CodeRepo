# ComputePixel

ComputePixel的主要功能是实现计算图片中的各个颜色的像素的数量。

## 使用方法

直接将代码中的**设置区**中的file变量设置为想要计算的图片的路径。然后Run这个代码就好啦！

## 源码环境依赖

开发环境为3.7.0。需要额外安装的第三方库只有**Pillow**，可直接使用pip安装。

```
pip install pillow
```
